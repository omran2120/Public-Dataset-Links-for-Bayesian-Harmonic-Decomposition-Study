# Public Dataset Links for Bayesian Harmonic Residual Fault Diagnosis Study

This repository provides the public dataset links used in the study:

**Bayesian Harmonic Decomposition for Machinery Fault Diagnosis with Uncertainty Quantification**

No new experimental data were generated in this work. The study uses publicly available reference datasets for bearing and gearbox vibration-based fault diagnosis. This repository is intended to support reproducibility by collecting the dataset sources used in the manuscript.

## Datasets

The manuscript uses the following public benchmark datasets:

1. Case Western Reserve University Bearing Data Center (CWRU)
2. Paderborn University Bearing Dataset
3. University of Ottawa Bearing Vibration Dataset
4. Jiangnan University Bearing Dataset (JNU)
5. Machinery Failure Prevention Technology Dataset (MFPT)
6. MCC5-THU Gearbox Fault Diagnosis Dataset

## Data Availability

All datasets are publicly available from their original providers. No dataset in this repository was generated by the authors. Users should cite the original dataset sources when reusing the data.

## Purpose

This page serves as a consolidated reference list for the datasets used to evaluate the Bayesian Harmonic Residual method, including diagnostic performance, uncertainty quantification, and reliability-oriented alarm calibration.
